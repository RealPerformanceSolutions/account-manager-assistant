# Changelog

All notable changes to this plugin. Bump `version` in `.claude-plugin/plugin.json` with each entry.

## [0.1.0] — Initial public release

A personal AI assistant for account managers, built around one rule: it prepares, you approve.

- `/setup` and `/edit-context` — interview-based configuration that scaffolds your workspace and
  `CLAUDE.md`.
- `/connect-outlook` — guided Microsoft 365 connect and verify.
- `/map-workflow` — map a real process (e.g. quote-to-PO) so the commands follow how you work.
- Day-to-day: `/morning-briefing`, `/draft-reply`, `/draft-forward`, `/overdue-quotes`,
  `/quote-request`, `/file-quote`, `/prep-calendar`, `/meeting-actions`, `/monthly-report`.
- Skills: `voice-writing` (style only, no personal data stored), `report-builder` (never invents
  figures).
- Templates: `email-reply`, `quote-request`, `monthly-report`, `account-notes`.
- Docs: an IT-approval email and a plain-English scope note.
